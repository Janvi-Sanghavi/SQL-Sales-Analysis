# SQL Sales Performance Analysis

## Project Overview

This project focuses on analyzing sales data using SQL to extract meaningful business insights.
The dataset was cleaned, transformed, and structured into relational tables to enable efficient querying and analysis.

---

## Tools & Technologies
* MySQL
* SQL (Joins, Aggregations)

---

## Dataset

* Superstore Sales Dataset (CSV format)

---

## Project Workflow

### 1. Data Cleaning

* Converted date formats using `STR_TO_DATE`
* Cleaned numeric values using `REPLACE`, `TRIM`, and `CAST`

### 2. Data Modeling

* Designed normalized tables:

  * Customers
  * Orders
  * Products
  * Sales

### 3. SQL Analysis

Performed analysis using:

* Joins
* Aggregations (`SUM`, `GROUP BY`)
* Filtering and sorting

---

## Key Insights

* Identified top-performing regions based on revenue
* Analyzed customer spending behavior
* Evaluated product category performance
* Observed sales trends over time

---

## Skills Demonstrated

* Data Cleaning & Transformation
* Database Design (Normalization)
* SQL Query Writing
* Analytical Thinking

---

## 💡 Conclusion

This project demonstrates how raw sales data can be transformed into actionable insights using SQL.
